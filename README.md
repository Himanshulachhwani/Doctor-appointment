# Doctor's Appointment booking system

# Project Overview and Features
This project is used to book a doctors appointment providing people features like the ability to book their own doctor or slot as per their choice and also while saving the appointments while the program is running, Which helps book follow-ups or check their appoointment status.
Project can also be used to save the patient's information for future use.
# Technologies used:
To Create this project, I have made use of classes, variables, loops, conditional statements,etc
# Installation:
To install the project, you can clone this repository using HTTP or SHH on to your local device and then run clinic.py
